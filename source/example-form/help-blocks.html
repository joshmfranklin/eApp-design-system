Please enter your entire first name. If your first name is a single letter, select the "Initial only" checkbox and type the letter below.

Enter a valid middle name.

Please enter your entire middle name.If your middle name is a single letter, select the "initial only" checkbox and type the letter below. If you don't have a middle name, select the "No middle name" checkbox and leave the field below blank.

Enter a valid last name

Please enter your entire last name. If your last name if a single letter, select the "Initial only" checkbox and type the letter below.

Use "Other" if your suffix isn't listed

If the suffix does not appear in this list, select "Other" and enter the suffix in the provided field.

List all other names you have used no matter how long ago

Input names one at a time, you will have the ability to add multiple other names.Example: Full name is Michael and nickname Mike. "Mike" would be the other name used.

Need help with the maiden name question?

Select "Yes" if this was your surname used from birth before it was legally changed at marriage.

Need help with the dates used range?

This date range is for the full time period you used the name or people knew you by it. Example: If the name was used from your birth until the present use your birthdate for "From" and select "Present"

Need help with the explanation?

We want to understand why this other name was used and/or changed. Examples: Nickname used since birth, changed when married, pen name.

Email addresses are not required

Email addresses are not required but may assist in the completion of your background investigation. Email format example: name@example.com

Enter only current phone numbers

Additional numbers provided may assist in the completion of your background investigation.

Check "Estimated" if date of birth is unknown

If you are not sure of your exact date of birth estimate it as best you can and check the "Estimated" checkbox.

Check "Not applicable" if no SSN

If a U.S. Social Security Number isn't present check the "Not applicable" checkbox.

Use feet and inches only

Feet must be a number between 1 and 9. Inches must be between 0 and 11.

Use pounds and whole numbers only

Weight must be a number between 10 and 999.
